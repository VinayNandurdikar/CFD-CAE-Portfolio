# Finite Difference Derivatives in Python

## Objective

This project demonstrates numerical differentiation using finite difference methods implemented in Python.

## Numerical Schemes

### Forward Difference

df/dx ≈ (f(x+h)-f(x))/h

### Backward Difference

df/dx ≈ (f(x)-f(x-h))/h

### Central Difference

df/dx ≈ (f(x+h)-f(x-h))/(2h)

## Applications in CFD

Finite difference methods form the basis of numerical discretization used in computational fluid dynamics.

These methods are used in:
- Gradient calculations
- Convection terms
- Diffusion terms
- PDE discretization

## Learning Outcomes

- Numerical approximation of derivatives
- Understanding truncation errors
- Comparison of numerical schemes
- Foundation of CFD discretization
