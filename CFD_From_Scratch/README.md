
# CFD From Scratch

This section contains Python implementations of numerical methods and CFD fundamentals.

## Purpose

The aim is to build CFD understanding from first principles by implementing:

- Finite Difference Method
- Numerical differentiation
- PDE discretization
- Stability and truncation error analysis
- Basic CFD solvers

## Roadmap

1. Finite Difference Derivatives
2. Second Derivative and Heat Conduction
3. 1D Linear Convection
4. Burgers Equation
5. 2D Diffusion
6. Lid-driven Cavity Flow
