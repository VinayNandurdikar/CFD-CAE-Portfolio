# CAE Parametric Optimization of a Mechanical Bracket

## Project Repository

[View full project on GitHub](https://github.com/VinayNandurdikar/CAE-Parametric-Optimization-Bracket)

## Overview

This project demonstrates a CAE-based parametric optimization workflow for a mechanical bracket. It includes geometry parameterization, meshing, structural analysis, deformation comparison, and parametric result evaluation.
